#### 接口名：获取待办详情
#### 接口描述：获取单个待办事项的详细信息，包含子任务、活动日志等
#### 接口地址：`/api/todo/detail`
#### 请求方式：`POST`
#### 请求头：
| 参数名称 | 参数说明 | 类型 | 是否必填 | 示例 | 长度限制 | 取值范围 |
|--------|--------|------|----------|------|------|------|
| Cookie | Cookie | string | 是 |  | 无 | 无 |
| Content-Type | json content type | object | 是 | application/json | 无 | 无 |

#### 请求参数类型：
`request_json`
#### 响应参数类型：
`response_json`
#### 请求参数：
| 参数名称 | 参数说明 | 类型 | 是否必填 | 示例 | 长度限制 | 取值范围 |
|--------|--------|------|----------|------|------|------|
| id | 待办任务id | string | 必填 | "123456" | 无 | 无 |

#### 请求示例：
```json
{ 
  "id": "123456"
}
```
