#### 接口名：待办列表
#### 接口描述：
#### 接口地址：`/api/todo/list`
#### 请求方式：`POST`
#### 请求头：
| 参数名称 | 参数说明 | 类型 | 是否必填 | 示例 | 长度限制 | 取值范围 |
|--------|--------|------|----------|------|------|------|
| Cookie | Cookie (Only:undefined) | string | 是 |  | 无 | 无 |
| Content-Type | json content type | object | 是 | application/json | 无 | 无 |



#### 请求参数：
| 字段名 | 类型 | 必填 | 描述 |
  |--------|------|------|------|
  | statusList | List<Integer> | 否 | 待办状态：0-待处理 1-已完成 2-进行中 3-已逾期 4-已取消 5-未接受 |
  | queryType | Integer | 否 | 查询类型：1-查我分配的待办 2-查我执行的待办 其他-不支持 |
  | deadLine | String | 否 | 截止时间 |
  | finishTime | String | 否 | 完成时间 |
  | projectId | String | 否 | 项目ID |
  | parentTodoId | String | 否 | 父待办事项ID |
  | umList | List<String> | 否 | 执行人UM账号 |
  | page | Integer | 否 | 当前页 |
  | pageSize | Integer | 否 | 分页大小 |

### 响应参数
响应格式为 ResultInfo 对象，包含以下字段：

#### ResultInfo 字段说明
| 字段名  | 类型   | 描述                     |
|---------|--------|--------------------------|
| code    | String | 响应码，200表示成功      |
| message | String | 响应信息                 |
| body    | Object | 响应数据体               |

#### body 字段说明 (QueryTodoResultDTO)
| 字段名称 | 类型             | 必填 | 描述                           |
|----------|------------------|------|--------------------------------|
| list     | List<TodoListDTO> | 否   | 待办任务列表                   |
| total    | Long             | 否   | 总数                           |
| page     | Integer          | 否   | 当前页                         |
| pageSize | Integer          | 否   | 分页大小                       |
| pageCount| Integer          | 否   | 页数                           |
